# Design Direction

## Style

- Clean SaaS
- Mobile-first
- Professional dashboard
- Không vibe code
- Giao diện sáng, dễ nhìn
- Spacing thoáng
- Card rõ ràng
- CTA nổi bật

## Inspiration

- shadcn/ui
- Linear
- Notion
- Stripe dashboard
- Shopify Polaris
- GitHub Primer

## Layout

- Landing page: hero, problem, solution, features, workflow, pricing, CTA
- App: sidebar trái, topbar, main content
- Dashboard: stat cards, AI suggestions, today's tasks, content table
- Forms: label rõ, helper text dễ hiểu
- Empty state: có hướng dẫn bước tiếp theo

## Colors

- Background: slate-50 / white
- Text: slate-900
- Muted: slate-500
- Border: slate-200
- Primary: blue / emerald
- AI Accent: violet / amber

## Rules

- Không copy y chang repo khác
- Chỉ tham khảo layout, spacing, UX pattern
- Code sạch, tách component hợp lý
- Ưu tiên mobile-first
